Hamiltonian

For this model a nearest neighbor approximation was used. The Hamiltonian of the system is calculated by summing all nearest neighbors interactions. Where A↔A, A↔B and B↔B refers to the interaction energy between these atoms.

States

The grid of atoms is represented as a 2d array at each position there is a either an atom A or an atom B. If Move Vacancy is chosen there is a single vacancy present in the grid.
